1.What is LWC?
  Lightning Web Components (LWC) is Salesforce's modern UI framework built directly on native browser web standards. It uses lightweight, standard web code to deliver exceptional performance without relying on heavy custom abstractions.
2.Why Salesforce Uses LWC
  Salesforce uses LWC because it runs significantly faster than older frameworks like Aura by executing natively in the browser. It also allows developers to build responsive interfaces using standard JavaScript, HTML, and CSS instead of proprietary languages.

4.Component Breakdown
An LWC bundle consists of an .html file for UI structure, a .js file for controller behavior, and a .js-meta.xml file to configure where the component can be placed. An optional .css file can be added to apply custom component-specific styling.
5.Frontend vs Backend Logic
Frontend logic (LWC) runs entirely in the user's browser to handle UI rendering, user inputs, and local screen behavior. Backend logic (Apex) executes securely on Salesforce cloud servers to perform database operations, record queries, and enforce security.
