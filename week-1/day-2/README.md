1. What is Salesforce Platform?
Salesforce Platform is a cloud-based system used to build and manage business applications. It allows companies to create apps quickly without worrying about servers or infrastructure.
2. Explain App, Object, Tab
App:
An App is a collection of features, objects, and tabs used for a specific business purpose like sales or service.
Object:
An Object is used to store data in Salesforce. It is similar to a table in a database. Examples are Account, Contact, and Opportunity.
Tab:
A Tab is used to access objects or apps in Salesforce. It helps users navigate and view data easily.
3. Difference: Configuration vs Coding
Configuration:
This is done using clicks in Salesforce setup. It does not require programming. Example: creating objects, fields, and workflows.
Coding:
This involves writing code like Apex or Lightning Web Components. It is used for complex logic that cannot be done through configuration.
4. System Design (Simple Understanding)
In a Salesforce Sales system:
The App is the Sales Application
Objects used are Account, Contact, and Opportunity
Users interact by:
Creating an Account for a company
Adding Contacts under that Account
Creating Opportunities for potential sales
