1. What is Flow Builder?
Flow Builder is a point-and-click tool in Salesforce used to create automation. It allows us to build complex business logic without writing any code.
2. Types of Flows
Screen Flow: A flow that requires user interaction. It guides the user through a series of screens to collect or display information.
Record-Triggered Flow: An automated flow that runs in the background whenever a record is created, updated, or deleted.
3. My Automation Ideas
Student Welcome: Auto-send an email once a student record is created.
Course Full Alert: Notify faculty when a course reaches its maximum capacity.
Birthday Greetings: Send an automated email to students on their birthday.
GPA Warning: Automatically flag a student record if their GPA falls below 2.0.
Fee Reminder: Send an alert 5 days before the tuition deadline.

6. Reflection: Why automation matters
Companies need automation because it eliminates human error and saves time. In a large system like a college, handling thousands of students manually is impossible. Automation ensures that the data is always accurate and the communication is always on time.
